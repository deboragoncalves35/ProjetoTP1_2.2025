Detailed Description

Representa um valor monetario entre 0,01 e 1.000.000,00.

O valor e armazenado como inteiro (em centavos) para evitar erro de arredondamento. O valor informado deve conter apenas digitos e, opcionalmente, ponto ou virgula.

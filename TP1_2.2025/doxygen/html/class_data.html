Detailed Description

Representa uma data no formato DD-MMM-AAAA.

O formato valido e DD-MMM-AAAA, onde o mes deve ser uma sigla em letras maiusculas (JAN, FEV, MAR, ABR, MAI, JUN, JUL, AGO, SET, OUT, NOV, DEZ). A data deve ser valida, considerando o ano bissexto.
